## 📌 Overview

SecureSys Frontend - A secure web-based messaging interface with end-to-end encryption, JWT authentication, and OTP verification. This is the client-side application that communicates with the SecureSys Backend API.

## ✨ Features

- 🔐 **JWT Authentication** - Secure token-based login
- 📧 **Secure Messaging** - Send/receive encrypted messages
- 🔑 **OTP Verification** - One-time password for viewing messages
- 📎 **File Attachments** - Upload and share encrypted files
- 👥 **Admin Dashboard** - User management and activity monitoring
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🎨 **Modern UI** - Gradient backgrounds, smooth animations
