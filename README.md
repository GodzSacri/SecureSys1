## 📌 Overview

SecureSys Backend API - A RESTful Flask API powering the SecureSys messaging platform. Features JWT authentication, end-to-end encryption, OTP verification, file uploads, and admin analytics.

## ✨ Features

- 🔐 **JWT Authentication** - Token-based authentication with expiration
- 🔒 **End-to-End Encryption** - AES-256-GCM for messages and files
- 📧 **OTP Verification** - One-time password via email
- 📎 **File Uploads** - Encrypted file storage
- 👥 **Admin Endpoints** - User monitoring and analytics
- 🗄️ **MySQL Database** - Persistent storage for users and messages
- 🚀 **CORS Enabled** - Cross-origin requests supported
